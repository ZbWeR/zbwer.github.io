  • 程序由什么组成：

​ 程序用于解决计算问题，所以程序的组成其实取决于问题的组成。我们先看上面这个问题由什么构成，上题题面中给出了\(D1,C,D2\)等数据，让我们计算费用，而费用的公式我们是知道的（怎么算油钱就不赘述了）。所以一个问题是由已知数据，计算目标，计算公式组成的。那么对于一个程序，首先要能把这些已知的数据输入到电脑中，再经过一系列的计算，最后把我们想要的答案输出来。

​ 总而言之，==一个基本的程序包括：输入数据，进行运算，输出答案。==

在介绍这三个东西之前，还有一点前置知识需要了解，我们看到一个简单的程序。

#include<bits/stdc++.h>
using namespace std;

int n,a,b,cnt,now,ans;

int main()
{
    scanf("%d",&n);
    scanf("%d",&now);cnt=1;
    for(int i=2;i<=n;i++)
    {
        scanf("%d",&a);
        if(a==now+1)
            cnt++;
        else
        {
            ans=max(ans,cnt);
            cnt=1;
        }
        now=a;
    }
    ans=max(ans,cnt);
    printf("%d",ans);
    return 0;
}

（\(PS:\)观察上面程序可以发现基本上每一行结束都由一个";"分号，分号在程序中的作用相当于语文中的句号）

我们来分块解读一下整个程序，对于前两行代码

这是一个程序的必不可少并且千篇一律的部分，基本上每个程序都要打这两行代码。

输入输出数据

  • 什么是数据？或者说数据有哪些类型？

​ 显然数字就是一种数据，再细分一些，数字包括整数，小数等等。其次字符也是一种数据，包括我们熟悉的"abc"这些英文字母，也包括感叹号问号这些特殊字符。

  • 数据输入到哪儿？输入给谁？

​ 首先，我们要有不同的容器来存放这些数据，就好比现在你知道2件衣服108元，你输入了2 和 108，计算机怎么知道2代表什么意思，108代表什么意思。就像应用题解方程一样，我们在开头总要说清楚，比如设花费的油钱为\(x\)元，那么此时\(x\)就相当于一个容器。所以在输入数据之前，我们首先要声明一些变量。比如说我们声明两个变量，一个叫做\(money\)，用来储存衣服的价格，一个叫做\(cloth\)用来储存衣服的件数。

基础的格式是：[变量的数据类型] [变量的名称]

运算

  • 怎样进行运算？

对于一个式子\(1+1\)，我们知道它的结果为\(2\),是因为我们知道\(+\)这个符号代表加法。同样，让计算机进行各种运算的时候，我们也需要类似于\(+-*/\)这样的运算符，让它知道要进行哪种运算。

  • 有哪些运算符？

运算符的分类：算术运算符、关系运算符、赋值运算符和逻辑运算符。

下面通过几个程序来详细解释这些不同的运算符。

//为书写方便，以下程序将省略一二两行，仅保留主函数部分
int main()
{
/*
算术运算符包括 + (加), - (减) , *(乘) , \(除) , %（取余）
对于前四种运算符，此处不再赘述。这里只介绍 % 取余（也称取模）运算的特点

注意：除法的使用与数学中不同，请自行探索

a%b=c 其中 c 是a除以b后所得的余数
例如：
5%2= 1 因为 5 =2*2+1 余数为1
19%5= 4 因为 19=5*3+4 余数为4
*/
   int a,b;
    scanf("%d %d",&a,&b);
    printf("%d %d %d",a+b,a-b,a%b);
    //输出a+b,a-b,a%b的值，自己试一下！
	return 0;
}
/*
赋值运算符： = （即等号）
对于式子 a=5; 表示的含义是，将5这个值交给a,即a这个变量中存储的数据是5
对于式子 a=b; 表示的含义是，将b这个变量的值交给a.注意此操作不会改变b的值
*/
int main()

分支结构

来看这样一个题目，

给定一个数n,判断n是偶数还是奇数，偶数输出"Yes",奇数输出"No"

我们发现，根据目前掌握的知识并不能解决。我们先用人话来描述一下解题思路，偶数的标准是能被2整除，意思就是这个数对2进行取模运算的结果是0.根据这一点，如果\(n\)%\(2=0\)那么我们就输出Yes,反之输出"No"就行了。

问题在于，这个中文”如果“在计算机中用什么来表示？

/*
介绍一个条件判断的语句结构

if(表达式1)
{
	表达式2;
}
表达式3;

它的作用是，如果表达式1为真命题，那么就会执行大括号里表达式2的内容，接着进行表达式3的内容
如果表达式1为假命题，就会直接执行表达式3的内容
*/
int main()
{
	int n;
    scanf("%d",n);//输入n
    if(n%2==0)//如果n对2的余数为0,即n是偶数
    {
        printf("Yes");//我们输出"Yes";
    }
    //下面这个if也可以不写，直接输出"No"也行
    if(n%2==1)
    {
        printf("No");
    }

	return 0;
}
/*
再介绍另一种结构，双分支结构

if(表达式1)
{
	表达式2;
}
else
{
	表达式3；
}

它的作用是：如果表达式1为真就执行表达式2,否则的话就执行表达式3
上述的程序可以改写为下面这种形式
*/
int main()
{
	int n;
    scanf("%d",n);
    if(n%2==0)
    {
        printf("Yes");
    }
    else
    {
        printf("No");
    }

	return 0;
}

我们再看到一个题目:

给定一个数n,如果n大于0,则输出1;如果n等于0，则输出0;如果n小于0,则输出-1

根据我们上面学的知识，不难得出这样一份代码

/*
如果if下的大括号内只有一个表达式,我们可以省略大括号不写
如if(a>b)
{
	printf("1");
}
可写为
if(a>b) printf("1");

*/
int main()
{
    int n;
    scanf("%d",n);
    if(n<0) printf("1"); 
    if(n==0) printf("0");
    if(n<0) printf("-1");
    return 0;
}

值得注意的是，我们不能写成这样

if(n<0) printf("1"); 

if(n==0) printf("0");
else  printf("-1");

因为\(else\)是和\(if\)配套使用的，此处的\(else\)表示的是：若\(n\)等于\(0\)不成立就输出\(-1\).

即\(else\)的否定是只针对上一个\(if\)而言的

接下来再介绍一个新的结构

/*
if(表达式1)
{
	表达式2;
}
else if(表达式3)
{
	表达式4;
}
else if(表达式5)
{
	表达式6;
}
else
{
	表达式7；
}
它的作用是：
如果1成立，就执行2.如果1不成立3成立，就执行4.如果3也不成立5成立,就执行6......
依此类推,我们可以写很多个else if.
最后到了else 即 如果上述的表达式都不成立，就执行7;
那么上述题目的代码可以改写为
*/
int main()
{
    int n;
    scanf("%d",n);
    if(n<0) printf("1"); 
    else if(n==0) printf("0");
    else printf("-1");
    return 0;
}

接下来我们通过一个例题来介绍上文提到的逻辑运算符

给定两个整数a,b.如果a能被2整除且a大于b,则输出Yes

这里涉及到了并且这个操作，让我们看看如何实现

/*
逻辑运算符的分类：
&& 表示并且的意思
|| 表示或者的意思
!  表示取反
*/
int main()
{
	int a,b;
    scanf("%d %d",a,b);
    if(a%2==0 && a>b) printf("Yes");
    //a能被2整除即取余2的结果为0
    
// a&&b只有当a,b都为真命题时才是真命题
// a||b只要有一个为真就是真命题
// !a 假如a为真命题，则!a 为假命题;反之a为假命题则!a为真命题
    return 0;

}

循环结构

我们知道计算机程序的特点之一就是进行重复的运算，那么如何实现呢？我们先来看一个题目

给定一个整数n,求1+4+9+16+....+n²的值

对于这种要进行多次运算的题目，我们就需要用到循环结构

/*
基本结构
for(初始化;条件判断;增量表达式)
{
	表达式1;
}
但从框架来看肯定是云里雾里的，我们直接看具体的情景下如何使用
*/
int main()
{
	int n,sum=0;//sum变量用于累加答案
    scanf("%d",n);
    
    for(int i=1;i<=n;i++)
    {
		sum=sum+i*i;
    }
    
     //for结构小括号里由两个分号分为三个部分，为便于表述记为1，2，3
    //第一个部分为初始化,在这个表达式中,我们定义了一个变量i,并令i的初始值为1
    //计算机首先执行初始化部分，然后进入到部分2(条件判断)
    //如果部分2的表达式为真，那么就会执行大括号里的内容，然后再执行部分3,然后再转到部分2进行判断，以此类推
    //如果为为假，则会跳出循环.即不再执行部分3和大括号里的内容
    
    //在下面的程序中，我们假设此时的n=5，首先初始化i=1,接下来进入部分2
    //我们发现2<=5为真,那么我们执行大括号里的内容，sum=sum+i*i;
    //即sum=sum+1*1=0+1=1，然后执行部分3
    //部分3的意思就是i=i+1,即i=i+1,此时i=2
    //然后转到部分2判断,有2<=5以此类推
    //直到i=6,此时不满足6<=5，则跳出循环，转为执行输出
    
    printf("%d",sum);
	return 0;
}
下面给出一些for循环的应用事例
    
    for(int i=1;i<=100;i=i+2)//即i=1-100中的所有奇数
	for(int i=2;i<=100;i=i*2)//即i=1-100中2的次幂
//需要注意的是，在for()的小括号里定义的变量，在结束for循环后就不再生效.

此题中\(sum\)这个变量的作用请好好体会

下面我们再看一道题,

题目描述：所谓谓角谷猜想，是指对于任意一个正整数，如果是奇数，则乘3加1，如果是偶数，则除以2，得到的结果再按照上述规则重复处理，最终总能够得到1。如，假定初始整数为5，计算过程分别为16、8、4、2、1。程序要求输入一个整数，将经过处理得到1的过程输出来。

这道题与上一道题的不同之处就在于，上一道题我们知道要计算\(n\)次就能得出答案，而这一道题我们不知道要算多少次.

/*
引入一个新的循环结构：
while(表达式1)
{
	表达式2;
}
它的运行过程是这样的，先判断表达式1是否为真，若为真，则执行表达式2.
执行完表达式2后再一次判断表达式1是否为真，若为真就执行表达式2,以此类推
直到表达式1不为真，直接跳出循环.
*/
int main()
{
	int n;
    scanf("%d",&n);
    
    while(n!=1)//依题意最终n=1,故n==1时才退出循环
    {
		if(n%2==0) n=n/2;//偶数
        else n=n*3+1;//奇数
        printf("%d",n);//输出变化过程
    }
	return 0;
}

总结：若已知运算次数，则适合使用\(for\)循环，若未知运算次数，则考虑使用\(while\)循环,还有一种\(dowhile\)循环请读者自行了解.下面给出一些常见的循环用法

//读入n次数据
for(int i=1;i<=n;i++)
    	scanf("%d",a);
//其他指令
for(int i=1;i<=n;i++)
{
	if(i==5) continue;
    //continue的作用是跳过接下来的表达式,直接进入i++环节
    else 表达式1;
}

函数

注：此部分内容笔者其实掌握程度并不深，有兴趣的可以点链接学习C语言函数完全攻略 (biancheng.net)

  • 什么是函数？

    函数就是具有独特功能的工具，比方说$S(r)=r^2 \(这个函数，它的功能就是知道半径可以求出边长,比方说\)max(a,b)$这个函数，就可以求出两个数中最大的那个数.

  • 怎样编写函数？函数由哪些部分组成？

    考察$S(r)=r^2 \(，我们发现函数包括函数的名字(\)S\(),函数包含的参数(\)r\(),以及函数的运算过程(\)r^2$)

    在使用一个函数之前，我们需要定义函数，一般格式为：
返回类型 函数名称(类型参数1，类型参数2,...)
{
	运算表达式;
}

//计算1+2+..+n的累和函数
int Sum(int n)
{
	int sum=0;
    for(int i=1;i<=n;i++)
        sum=sum+i;
   	return sum;
}
/*
这个函数的作用是计算1加到n的和，最终的答案应该是整数，即我们的返回类型应该设置为int
(int n)代表的参数，该函数只有n一个参数，并且n是整数，所以写为int n
中间部分为具体运算过程，return sum 表示返回最终的答案
*/

//求小数a,b中的最大值
double Max(double a,double b)//多个参数之间,用逗号连接
{
	if(a>b) return a;
    else return b;
}
//最终答案是double类型的数据，所以返回类型应该设置为double

//无返回值函数
void Pint(int a)

	for(int i=1;i<=a;i++)
        	printf("*");
}
